# Jogo da Cobrinha

## Descrição
Este é um jogo clássico da cobrinha (Snake Game) desenvolvido com HTML, CSS e JavaScript. O jogador controla uma cobrinha que deve comer alimentos para crescer, evitando colidir com as paredes ou com o próprio corpo.

## Funcionalidades
- Controle da cobrinha usando as teclas de seta
- Sistema de pontuação
- Aumento da velocidade conforme a pontuação aumenta
- Interface amigável com botões para iniciar e reiniciar o jogo
- Feedback visual quando o jogo termina

## Como Jogar
1. Abra o arquivo `index.html` em seu navegador
2. Clique no botão "Iniciar Jogo"
3. Use as teclas de seta para controlar a direção da cobrinha
4. Tente comer o máximo de alimentos possível sem colidir com as paredes ou com o próprio corpo
5. Quando o jogo terminar, clique em "Reiniciar" para jogar novamente

## Tecnologias Utilizadas
- HTML5 (Canvas para renderização do jogo)
- CSS3 (Estilização e layout)
- JavaScript (Lógica do jogo)

## Estrutura do Projeto
- `index.html` - Estrutura da página e elementos do jogo
- `style.css` - Estilização dos elementos
- `script.js` - Lógica do jogo

## Personalização
Você pode personalizar o jogo alterando as seguintes variáveis no arquivo `script.js`:
- `gridSize` - Tamanho de cada célula do grid
- `speed` - Velocidade inicial da cobrinha
- `snakeColor` - Cor da cobrinha
- `foodColor` - Cor da comida

E no arquivo `style.css` você pode alterar as cores, tamanhos e estilos dos elementos da interface.
